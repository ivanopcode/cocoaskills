CocoaSkills

Local skill manager for AI agent skills with reproducible per-project installs.

curl -fsSL https://cocoaskills.org/install.sh | sh

Or install with mise:

mise use -g pipx:cocoaskills@latest

Source, documentation, and releases live at github.com/ivanopcode/cocoaskills.

Docs: skill authoring guide, RFC 0007 skill dependencies (ru), MVP design, RFC 0003 runtime roots.
